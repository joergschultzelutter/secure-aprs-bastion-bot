# `--delete-command` - deletes a command code/command string from the config file

## Introduction

Deletes a [`--command-code`](add-command.md#--command-code) / [`--command-string`](add-command.md#--command-string) combination from the config file (whereas present)

## Description

## Example
```
python configure.py --delete-command --callsign=DF1JSL-1 --command-code=sayhello
```
